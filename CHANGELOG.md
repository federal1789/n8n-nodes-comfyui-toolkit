# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-25

### Added
- **ComfyUIUploadImage** — Upload images to ComfyUI for use in workflows
- **ComfyUITextToImage** — Generate images from text prompts via ComfyUI
- **ComfyUIImageToImage** — Transform existing images using ComfyUI workflows
- **ComfyUITextToVideo** — Generate videos from text prompts via ComfyUI
- **ComfyUIImageToVideo** — Animate images into videos using ComfyUI
- **ComfyUIWaitUntilResult** — Poll ComfyUI until a job completes
- **ComfyUIGetResults** — Retrieve output files from a completed ComfyUI job
- CI pipeline with lint, build, and test checks on Node.js 18 and 20
- Automated npm publish workflow triggered on version tags
